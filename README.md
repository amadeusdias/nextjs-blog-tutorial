## This small blog is a tutorial project found in the Next.js documentation. It can be found at [Learn Next.js](https://nextjs.org/learn). 
### Here, basic concepts of the Next.js framework are presented, such as:
+ Page navigation
+ Different ways to integrate CSS
+ Data Fetching
+ How to use the SSR (Server Side Rendering) and Static Rendering features provided by the framework
+ Dynamic routes and their details
+ API Routes
+ How to Deploy using Vercel
